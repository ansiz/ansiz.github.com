CentOS下的各种折腾

发表于 2015-08-12   |   分类于 Linux   |  


由于新项目需要在linux开发，于是我弄了一个CentOS6.5。使用Linux的过程才觉得以前真的是被微软宠坏了啊，尤其是这种不是最新版的CentOS，各种开发环境和软件的部署或多或少都会遇到一些坑。把这些坑都写出来，在强化巩固自己记忆的同时希望能给大家带来一点帮助。

一、CentOS6.5下输入法的安装

其实正常情况下，用系统自带的输入法就好了，ibus是centos默认推荐的，直接在System–Preference–input method中设置输入法就好了。但是…如果你想要在SublimeText中输入中文的话…那么你需要安装fcitx才行。CentOS6.5安装fcitx需要先安装依赖包，fcitx的依赖包真心多：

yum install gettext gettext-devel libXft libXft-devel libXpm libXpm-devel automake autoconf libXtst-devel gtk+-devel gcc zlib-devel libpng-devel gtk2-devel glib-devel

是的就是这么多…有可能一次性安装会失败，那就逐个手动安装（我人品比较差反正一次性安装是没成功的后来手动安装了没装上的）

然后去下载3.x版本的fcitx，4.x的版本装不上。然后输入下面的命令：

$ wget http://fcitx.googlecode.com/files/fcitx-3.6.4.tar.gz  
$ tar zxvf fcitx-3.6.4.tar.gz  
$ cd fcitx-3.6.4  
$ ./autogen.sh  
$ mkdir /usr/local/fcitx  
$ ./configure --prefix=/usr/local/fcitx  
$ make && make install
接下来输入 
$ /usr/local/fcitx/bin/fcitx -h
如果出现下列信息就说明安装成功了： 
Usage: fcitx [OPTION]  
 -d        run as daemon(default)  
 -D        don't run as daemon  
 -c        (re)create config file in home directory and then exit  
 -n[im name]    run as specified name  
 -v        display the version information and exit  
 -h        display this help and exit

接下来配置fcitx输入法

1、创建fcitx输入法文件：

# vim /etc/X11/xinit/xinput.d/fcitx.conf  
XIM=fcitx  
XIM_PROGRAM=/usr/local/fcitx/bin/fcitx  
XIM_ARGS=”-d”  
GTK_IM_MODULE=xim  
QT_IM_MODULE =fcitx

2、设置fcitx输入法文件为默认

[root@localhost ~]$ mv /etc/alternatives/xinputrc /etc/alternatives/xinputrc.bak         //原来备份一下  
[root@localhost ~]$ ln -s /etc/X11/xinit/xinput.d/fcitx.conf /etc/alternatives/xinputrc  //设置fcitx为默认输入法

3、设置个人环境变量

[zxh@localhost ~]$ vim ~/.bashrc       //注意用户，最好是常用登录用户。修改一下内容  
# User specific aliases and functions  
# export GTK_IM_MODULE=ibus               //这是以前的  
# export XMODIFIERS=@im=ibus  
# export QT_IM_MODULE=ibus  
#export LANG="zh_CN.UTF-8"                 //如果系统字符不是中文的话，这个要加，不然乱码  
export XIM=fcitx                          //这是新的  
export GTK_IM_MODULE=xim  
export XMODIFIERS=@im=fcitx  
export QT_IM_MODULE=fcitx
