前端开发利器--SublimeText插件

发表于 2015-08-07   |   分类于 工具 ， SublimeText插件   |  


做前端开发的话，相信SublimeText肯定不陌生了，而SublimeText很强悍的一点正是丰富的插件。以SublimeText3为例介绍一下几个插件。

插件的安装步骤：
1、安装package control组件
在SublimeText中Ctrl+`（就是Tab键上方那个符号），在弹出的输入框中复制粘贴以下代码然后回车：

import urllib.request,os; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), ‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘,’%20’)).read())

稍等片刻，组件安装完毕，重启SublimeText，接下来Ctrl+Shift+P，在输入框中输入install Package回车，在列表中输入想安装的插件回车即可安装。

插件推荐：

如果js代码写得比较多的话，在调整缩进和保持代码整洁上面应该会浪费不少时间。JSFormat就是用来干这种活儿的，只需要按一下快捷键Ctrl + Alt + F代码马上就变得清晰了。JSFormat是一款在SublimeText2和3中都有的插件，在软件中安装好插件之后，找到JSFormat的配置文件，然后将”jslint_happy”: false,修改为”jslint_happy”: true，然后保存。这样子匿名函数会自动有空格缩进。

HTML-CSS-JS Prettify，安装后ctrl+shift+H就能格式化HTML和CSS的代码了，这里需要注意一下，这个插件依赖于node.js，所以一定要先配置一下node.js，Windows和linux下的node.js的配置安装都很简单，就不赘述了。

另外还有：

Sublime Prefixr
Prefixr，CSS3 私有前缀自动补全插件

SublimeLinter
一个支持lint语法的插件，可以高亮linter认为有错误的代码行，也支持高亮一些特别的注释，比如“TODO”，这样就可以被快速定位。

Hex to HSL
自动转换颜色值，从16进制到HSL格式，快捷键 Ctrl+Shift+U

Git
这个就不说了，实现了大多数git的功能
